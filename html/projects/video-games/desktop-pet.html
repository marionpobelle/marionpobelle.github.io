OVERVIEW

A little friend that runs around on your desktop, plays with your cursor and keeps you company. Made using Unity,  works with both Windows 10 and Windows 11. 

I had been thinking about getting a desktop pet for a while, but could not find anything that suited what I wanted, so I made my own!

My tasks included: :

  • Develop gameplay features (interactions…)
  • Develop and refine AI behavior (state machine, illusion of credible behavior…)
  • Implement User Interface (accessibility settings, scalability…)
  • Asset integration (User Interface, 2D sprites, Animations…)
  • Transparent background with the Windows API and compatibility
  • And more!

SEE-THROUGH BACKGROUND

In order to implement the see-through background and ensure that the user could interact with anything on their desktop outside of the application, I used the Unity Windows API.

To make the background see-through, I manipulated the application’s window layered attribute by adjusting the color key and alpha values for selective transparency, without making the window input transparent.

AI

The AI has a base class called AnimalHandler that takes care of anything relating to the AI’s mood, status, affection level, hunger, data saving, and basic interactions such as feed and drag. 

Then, each animal derives from this class using inheritance and implements its specific behaviors with a simple state machine, tuned to each animal’s needs.
