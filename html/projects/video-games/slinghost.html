OVERVIEW

Slinghost is an arcade shooter game with a physical installation, in which you have to shoot pumpkins with a giant slingshot at invisible ghosts, slowly coming towards you... before they reach you!

This project was realized in 5 days as part of a workshop on accessibility. 

NOTE : even though the game is downloadable and playable with mouse and keyboard, it was meant to be played using the homemade "giant slingshot controller" the team crafted, exposed during a temporary event.

The main elements that needed to be addressed were the arduino, the slingshot controller and the gameplay. My tasks included: 

  • Develop gameplay features (gameplay loop, AI…)
  • Implement User Interface (accessibility settings, scalability…)
  • Asset integration (User Interface, 2D sprites, animations, feedback…)
  • Working on the homemade controller (aim, haptic rumbles, construction, maximum stretch…)
  • And more!

CONTROLLER

The first main component of our controller is a Switch Joy-Con. It was easy to hook up to Unity and the Ring Fit shape was great to grip for a big Frankenstein slingshot. However, it had some shortcomings: the Joy-Con could only record its velocity and relative rotation, which was insufficient for accurate targeting.

Our second component, a VR set-up, covered for the Joy-Con limitations. We sat up the headset as well as lateral cameras around our installation and hooked up a VR controller to the Ring Fit. This way, we could access proper positioning and rotation data, allowing us to implement suitable aiming.

The whole contraption was tied up to a hand-crafted metal frame with robust elastics, as well as some regular ropes to ensure the elastics would not be stretched too hard.

When the velocity of the Switch Joy-Con reaches a certain threshold, we know the slingshot was launched, and we compute the impact point from the Joy-Con’s velocity for force, and from the position and rotation data of the VR controller.

A WII controller would have been a fun option to explore as well!

ACCESSIBILITY

The game was thought to be playable by people with visual or/and audio impairing disabilities. 

Within the game, visual, audio and haptic feedback allow the player to figure out the ghost's position. 

Outside of the game, the crafted slingshot controller has an adjustable height,  allowing people in wheelchair to play, and a "one-button" mode (to facilitate inputs) has been prototyped but not fully developed.
