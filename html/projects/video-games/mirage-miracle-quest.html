OVERVIEW

Due to NDA, I can’t actually reveal elements about Mirage Miracle Quest publicly, but I’ll still list out my responsibilities here as best I can.

Mirage Miracle Quest is a hand painted game offering a modern take on the classic turn-based rpg with deck-building elements and the magical girl trope, all set in a colorful retro world.

My duties included:

  • Develop gameplay features (player controller, combat, exploration, interactions…)
  • Implement User Interface (design menus, set up navigation…)
  • Asset integration (User Interface, 2D assets, audio, animations, objects…)
  • Implement localization and update the system and localized elements
  • Improve the team’s work environment with editor programming and a custom inspector
  • Performance control and optimization
  • Collaborating closely with environment artists, sound designers, narrative designers and game designers
  • Maintain a feature list with regular updates so that everyone can track progress
  • Writing technical documentation (naming convention, integration procedure…)
  • Implement a Vertical Slice!
  • Implement a Demo!
  • And more!
